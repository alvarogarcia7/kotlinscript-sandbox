# Kotlinscript demo

Run kotlinscript

## Running it

### Execute the script

On my system:

```bash
./validate-xml.kt mutation.xml
```

### Edit it

```bash
kscript --idea validate-xml.kt
```

(Remember to import the project in Idea)
